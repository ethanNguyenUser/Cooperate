# Two Squares: Co-Op Scroller

A cooperative scrolling shooter game where two players control different squares with unique abilities, sharing a single health bar.

## Controls

**Red Player (Larger, Slower):**
- Movement: <kbd>W</kbd><kbd>A</kbd><kbd>S</kbd><kbd>D</kbd>

**Blue Player (Smaller, Faster):**
- Movement: <kbd>Arrow Keys</kbd>
- Auto-fires towards Red player.

**Cooperative Abilities:**
- **Combined Attack:** <kbd>Space</kbd> (when close to each other, 8s cooldown) - Fires a powerful bullet.
- **Leash Damage:** <kbd>Shift</kbd> (4s cooldown) - Damages enemies touching the leash between players.

**General:**
- Start Game: <kbd>Enter</kbd>
- Restart Level: <kbd>R</kbd>
- Toggle Sound: <kbd>M</kbd>
- Select Level: <kbd>L</kbd>
- Cancel Level Select: <kbd>Esc</kbd>

## Gameplay

Players navigate a scrolling map, clearing all enemies to advance to the next level. Beware of different enemy types, ricochet turrets, and environmental obstacles. Collect power-up stars to gain temporary advantages like invincibility, bouncing bullets, speed boosts, spread shot, rapid fire, or health regeneration.

## Difficulty Levels

- **Easy**: 50% slower enemy movement, firing, and bullet speed
- **Medium**: 30% slower enemy movement, firing, and bullet speed  
- **Hard**: Base difficulty (default)
